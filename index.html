Shogi board with Vite+PreAct

This is an HTML page to exercise the TypeScript Shogi Board app. The app can render multiple Shogi Boards on a single page. More than two dozen ShogiBoard with various configurations are shown on this page.

The page is loaded with the Shogi data array with each element using different parameters for a various board setup. See readme for available options.

To simplify the page arrangement, I use BootStrap on this page. However, the app itself is not dependent on any CSS framework.

Cycling through data set

Event triggered prepared data loading

Using plain vanilla display: flex property to arrange two columns per row, the ShogiBoard on the left side and some verbiage text on the right side. First ShogiBoard data should appear to the left of this text or above, depending on the screen size

Bootstrap row-column setting is used for displaying board and article side by side. Using class="large" to display larger board.

Click on the right half of the board to advance one move. Click on the left half of the board to rewind one move.

Right click anywhere on the board to display the context menu for more actions.
